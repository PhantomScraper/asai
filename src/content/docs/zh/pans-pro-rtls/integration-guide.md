<h1>集成指南</h1>
<p>以下页面描述了 <a class="reference internal" href="/docs/zh/pans-pro-rtls#pans-pro-rtls"><span class="std std-ref">PANS PRO RTLS</span></a> DWM1001模块的公共应用程序接口。该模块为用户提供了多个与系统交互的界面。这些接口是 <a class="reference internal" href="/docs/zh/pans-pro-rtls/integration-guide/uart-interface#pans-uart-interface"><span class="std std-ref">UART</span></a>、<a class="reference internal" href="/docs/zh/pans-pro-rtls/integration-guide/spi-interface#pans-spi-interface"><span class="std std-ref">SPI</span></a> 和 <a class="reference internal" href="/docs/zh/pans-pro-rtls/integration-guide/ble-inteface#pans-ble-interface"><span class="std std-ref">BLE</span></a> 接口，可通过智能手机应用程序使用。</p>
