<span id="id1"></span><h1>PANS PRO UWBS</h1>
<p><a class="reference internal" href="#pans-pro-uwbs"><span class="std std-ref">PANS PRO UWBS</span></a> 是一个完整的实时定位系统和网络堆栈 - 可配置为锚点, 标签或关接节点。</p>
<hr class="docutils">
<div class="section" id="key-features">
<h2>主要功能</h2>
<ul class="simple">
<li><p>采用 TWR（双向测距）技术： 集成的定位引擎可让设备在导航模式下使用双向测距技术独立运行。</p></li>
<li><p>支持网络配置文件 0：完全可扩展的 <a class="footnote-reference brackets" href="#note-1" id="id2">1</a> 高容量, 低功耗。</p></li>
<li><p>支持网络可视化工具： 兼容 Android 版 PANS PRO Manager，以及 Windows, Linux 和 macOS 平台的 LEAPS Center WebApp。</p></li>
<li><p>固件更新与维护: 方便更新和持续维护。</p></li>
<li><p>兼容性： 与 PANS RTLS（免费版本）完全兼容，并包含长期的错误修复支持。</p></li>
<li><p>卓越的电源管理： 确保延长电池寿命。</p></li>
</ul>
</div>
