<h1>DWM1001C</h1>
<p>DWM1001模块基于Decawave的DW1000超宽带 (UWB) 收发器 IC，该 IC 是 IEEE 802.15.4-2011 UWB实现. 它集成了 UWB 和Bluetooth® 天线, 所有射频电路, Nordic Semiconductor nRF52832和运动传感器。</p>
<div class="admonition note">
<p class="admonition-title">注解</p>
<p>要获取更多信息，请访问 <a class="reference external" href="https://www.qorvo.com/products/p/DWM1001C">Qorvo的 DWM1001C 页面</a>。</p>
</div>
<hr class="docutils">
<div class="section" id="key-features">
<h2>主要功能</h2>
<ul>
<li><p>测距精度在10厘米以内。</p></li>
<li><p>UWB 频道 5 印刷电路板天线 (6.5 GHz)</p></li>
<li><p>6.8 Mbps 数据传输速率，符合 IEEE 802.15.4-2011 UWB 标准。</p></li>
<li><p>北欧半导体 nRF52832</p></li>
<li><p>Bluetooth®连接</p></li>
<li><p>Bluetooth®芯片天线</p></li>
<li><p>运动传感器 三轴加速计</p></li>
<li><p>为低功耗睡眠模式优化的电流消耗： &lt;15μA</p></li>
<li><p>电源电压：2.8 V 至 3.6 V</p></li>
<li><p>尺寸: 19.1 mm x 26.2 mm x 2.6 mm</p></li>
<li><p>标记为DWM1001C的模块已通过 ETSI, FCC 和 ISED 法规认证。</p>
<blockquote>
<div><ul class="simple">
<li><p>FCC ID：2AQ33-DWM1001，ICC：23794-DWM1001.</p></li>
<li><p>请参阅本数据表的模块标记部分。</p></li>
</ul>
</div></blockquote>
</li>
</ul>
</div>
<div class="section" id="software-compatibility">
<h2>软件兼容性</h2>
<p>它与 <a class="reference internal" href="/docs/zh/leaps-rtls#leaps-rtls"><span class="std std-ref">LEAPS RTLS</span></a> 和 <a class="reference internal" href="/docs/zh/pans-pro-rtls#pans-pro-rtls"><span class="std std-ref">PANS PRO RTLS</span></a> 都兼容。</p>
</div>
