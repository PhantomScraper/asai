<span id="welcome"></span><h1>欢迎阅读LEAPS文档！</h1>
<p>欢迎来到 <a class="reference external" href="https://www.leapslabs.com/">LEAPS</a> 的官方文档。在这里，您将找到 <a class="reference internal" href="/docs/zh/udk#udk-demo-kit"><span class="std std-ref">UDK DEMO KIT</span></a>、<a class="reference internal" href="/docs/zh/leaps-rtls#leaps-rtls"><span class="std std-ref">LEAPS RTLS</span></a> 和 <a class="reference internal" href="/docs/zh/pans-pro-rtls#pans-pro-rtls"><span class="std std-ref">PANS PRO RTLS</span></a> （实时定位系统）的每个组件的全面详细描述。我们的文档是一种宝贵的资源，为您提供了有效理解和利用我们系统所需的信息。</p>
<p>除了文件，我们很高兴提供 <a class="reference external" href="https://forum.leapslabs.com/">LEAPS Forum</a>，这是我们全球客户和用户社区的专用平台。这个论坛允许您参与讨论、提出疑问，并深入了解系统的复杂性。这是一个协作环境，您可以在其中与其他用户联系，并从共享的知识和经验中受益。</p>
<p>我们鼓励您查阅我们的文件并积极参与 <a class="reference external" href="https://forum.leapslabs.com/">LEAPS Forum</a>，以最大限度地发挥我们系统的潜力。</p>

<h2>文件的组织</h2>
<p>我们将文档分为六个部分：</p>
<ul class="simple">
<li><p><a class="reference internal" href="/docs/zh/leaps-solutions#leaps-solution"><span class="std std-ref">LEAPS 解决方案</span></a> 部分提供了有关LEAPS提供的产品和 Ultra-Wideband 实时定位系统的简明信息。</p></li>
<li><p><a class="reference internal" href="/docs/zh/udk#udk-demo-kit"><span class="std std-ref">UDK DEMO KIT</span></a> 部分提供了全面的细节，并便于使用 Ultra-Wideband 多功能演示即用套件轻松开始。</p></li>
<li><p><a class="reference internal" href="/docs/zh/leaps-rtls#leaps-rtls"><span class="std std-ref">LEAPS RTLS</span></a> 部分深入探讨了先进的 Ultra-Wideband 导航和跟踪系统LEAPS RTLS（实时定位系统）。</p></li>
<li><p><a class="reference internal" href="/docs/zh/pans-pro-rtls#pans-pro-rtls"><span class="std std-ref">PANS PRO RTLS</span></a> 一节深入探讨了对传统 Ultra-Wideband 导航和跟踪系统 PANS PRO RTLS（实时定位系统）的理解。</p></li>
<li><p><a class="reference internal" href="/docs/zh/faq#common-faq"><span class="std std-ref">常问问题</span></a> 部分提供了常见问题（faq）的答案。</p></li>
<li><p><a class="reference internal" href="/docs/zh/support#technical-support"><span class="std std-ref">支持</span></a> 部分是一个宝贵的资源，提供技术支持信息，以确保随时提供援助和指导。</p></li>
</ul>

<div class="section" id="about-us">
<h2>关于我们</h2>
<ul class="simple">
<li><p><a class="reference external" href="https://www.leapslabs.com/">LEAPS</a> 是一家总部位于布拉格（捷克共和国，欧盟）的公司，专注于构建和营销基于超宽带的实时定位和实时遥测系统。</p></li>
<li><p>LEAPS在使用超宽带进行系统开发方面拥有10多年的经验，是将超宽带芯片转化为有用解决方案方面最具创新性的团队之一。</p></li>
<li><p>我们的团队由来自布拉格、越南胡志明市和巴西Pelotas的所有高级嵌入式工程师组成，他们充满热情地工作。</p></li>
<li><p><a class="reference external" href="https://www.leapslabs.com/">LEAPS</a> 是流行的DWM1001C和PANS RTLS的创建者。</p></li>
<li><p><a class="reference external" href="https://www.leapslabs.com/">LEAPS</a> 一直专注于提供RTLS硬件和软件、设计服务和技术许可。</p></li>
<li><p><a class="reference external" href="https://www.leapslabs.com/">LEAPS</a> RTLS是一组高级软件，为各种用例提供位置服务。它提供了市场上最先进的全嵌入式UWB子系统，涵盖了从接近、导航和跟踪到实时数据遥测和路由回程的功能。想象一个GPS模块，但使用超宽带，功能更多。</p></li>
<li><p><a class="reference external" href="https://www.leapslabs.com/">LEAPS</a> 提供了一个新的维度，帮助企业定位新价值，提高流程效率，增加安全性，提高可靠性，在室内导航，流程自动化。。。</p></li>
</ul>
