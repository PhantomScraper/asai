<div class="section" id="software-infrastructure">
<h2>软件基础设施</h2>
<p>支持三个平台: VMWare, Docker和Raspberry Pi. 这些平台为运行各种应用程序提供了灵活性和兼容性. 只需几个简单的安装步骤，您就可以享受 <a class="reference internal" href="/docs/zh/udk/udk-start/twr-rtls-and-data-telemetry-demo#twr-rtls-and-data-telemetry-demo"><span class="std std-ref">TWR RTLS 和数据遥测演示</span></a> 或 <a class="reference internal" href="/docs/zh/udk/udk-start/uplink-tdoa-rtls-demo#uplink-tdoa-rtls-demo"><span class="std std-ref">上行链路 TDoA RTLS 演示</span></a>。</p>
