<span id="id1"></span><h1>leaps_stnry_cfg_set</h1>
<p>写入标签节点进行静态检测的配置。 即使静态检测被禁用，也可以写入配置（参见 <a class="reference internal" href="/docs/zh/leaps-rtls/integration-guide/api-commands/leaps-cfg-tag-set#leaps-cfg-tag-set"><span class="std std-ref">leaps_cfg_tag_set</span></a>）。 写入的是内部非易失性存储器，因此使用时应小心。 如果启用了静态模式，新的灵敏度设置会立即生效。 默认灵敏度为 “高”。</p>
