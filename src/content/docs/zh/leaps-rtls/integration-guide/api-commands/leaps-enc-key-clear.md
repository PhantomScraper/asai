<span id="id1"></span><h1>leaps_enc_key_clear</h1>
<p>清除加密密钥并禁用加密选项如果已启用。 如果密钥未设置，则不起任何作用。</p>
<p>此调用会在设置新值时写入内部闪存，因此不应频繁使用，最坏情况下可能需要数百毫秒！需要重置新配置才能生效。</p>
