<span id="id1"></span><h1>电源选项</h1>
<p>UDK设备提供两种可行的电源选项：USB-C电缆或3.7V电池电源。</p>
<a class="reference internal image-reference" href="../../../_images/powering_by_usb_battery.png"><img alt="../../../_images/powering_by_usb_battery.png" class="align-right" src="/docs-assets/zh-cn/_images/powering_by_usb_battery.png" style="width: 171.20000000000002px; height: 282.40000000000003px;"></a>
<p><strong>USB类型C</strong></p>
<p>每个设备都有两个 <a class="reference internal" href="/docs/zh/udk/udk-start/device-hw-interfaces#hardware-interface"><span class="std std-ref">USB-C端口</span></a>. 两者都被设计为电路板的电源输入端口. 电源可以来自PC或5V适配器. 如果需要，可以同时使用这两个 <a class="reference internal" href="/docs/zh/udk/udk-start/device-hw-interfaces#hardware-interface"><span class="std std-ref">USB-C端口</span></a>。</p>
<p><strong>电池</strong></p>
<p>此外，该设备可以使用电池供电，从而提供便携式电源解决方案. 如果需要，该设备可以同时由电池和 <a class="reference internal" href="/docs/zh/udk/udk-start/device-hw-interfaces#hardware-interface"><span class="std std-ref">USB-C端口</span></a> 供电</p>
