<span id="udkdemos"></span><h1>Demos</h1>
<p>一旦准备好 <span class="red-text">硬件</span> 和 <span class="red-text">所需工具</span>，本节将提供更详细的设置每个演示的全面指南。</p>
<p>它将涵盖以下几个方面：</p>
<ul class="simple">
<li><p><strong>所需硬件和软件</strong>：每个演示的要求和信息的简要列表。</p></li>
<li><p><strong>节点或网络配置</strong>:设备设置和配置，以实现所需的演示。</p></li>
<li><p><strong>运行基本演示</strong>:执行演示主要功能的快速指南。</p></li>
<li><p><strong>技术评估</strong>:用户使用演示设置评估技术的指南和提示。</p></li>
<li><p><strong>高级选项</strong>:用户访问更高级功能的指南和提示，以及评估后下一步的建议。</p></li>
<li><p><strong>故障排除</strong>:故障排除指南。</p></li>
</ul>
<p>请选择一个演示，并按照以下说明快速设置并准备进行评估。</p>
