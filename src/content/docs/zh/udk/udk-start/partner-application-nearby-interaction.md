<span id="anchor-ni-app"></span><h1>Qorvo 附近的互动</h1>
<p>本节提供了 <a class="reference external" href="https://apps.apple.com/tr/app/qorvo-nearby-interaction/id1615369084">Qorvo Nearby Interaction的所有必要安装和设置步骤</a> 应用程序. 配备U1芯片的iPhone（型号：iPhone 11, iPhone 12和iPhone 13）需要iOS 16。</p>
<div class="section" id="installation-instructions">
<h2>安装说明</h2>
<p>从iOS 16开始，引入了带有第三方配件的背景模式的近距离交互. 需要将iPhone更新到iOS 16版本或更高版本. <a class="reference external" href="https://apps.apple.com/tr/app/qorvo-nearby-interaction/id1615369084">Qorvo附近互动</a> iOS应用程序可在苹果应用商店下载：</p>
