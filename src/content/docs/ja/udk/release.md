<span id="udk-releases"></span><h1>リリース</h1>
<p>UDK デモキットのリリース ページへようこそ。ここでは、重要なサブセクションを含む、UDK デモキットの各リリースに関する詳細情報をご覧いただけます：</p>
<ul class="simple">
<li><p><strong>リリース日</strong>: 各リリースの日付が記載されており、UDK Demo Kit の最新アップデートを追跡できます。</p></li>
<li><p><strong>サポートされるバージョン</strong>： UDK Demo Kit の各リリースでサポートされるソフトウェアおよびハードウェアのバージョンに関する情報を提供します。</p></li>
<li><p><strong>新機能</strong>： UDK Demo Kit の各リリースで導入された新機能や拡張機能を紹介し、最新の機能を活用できるようにします。</p></li>
<li><p><strong>既知の制限事項</strong>: UDK Demo Kit に関連する既知の制限事項や制約事項の概要を説明します。これらの制限を理解することで、キットの使用方法を最適化できます。</p></li>
<li><p><strong>既知の問題点</strong>： UDK Demo Kit のパフォーマンスや機能に影響を及ぼす可能性のある既知の問題点やバグを紹介します。該当する場合は、回避策に関する情報も提供しています。</p></li>
</ul>
