<span id="limitation-1"></span><h1>LIMIT001</h1>
<p><strong>概要</strong></p>
<p>デバイスの UWB キャリブレーションはサポートされていません。</p>
<hr class="docutils">
<p><strong>ユーザーへの影響</strong></p>
<ul class="simple">
<li><p>なし</p></li>
</ul>
<hr class="docutils">
<p><strong>再現するためのステップ</strong></p>
<ul class="simple">
<li><p><a class="reference internal" href="/docs/ja/leaps-rtls/integration-guide/leaps-uwbs#leapsuwbs"><span class="std std-ref">LEAPS UWBS</span></a> の <strong>Qorvo UCIモード</strong> を設定してください。</p></li>
<li><p>予期しない動作です： デバイスの UWB キャリブレーション・パラメータはサポートされていません。</p></li>
</ul>
<hr class="docutils">
<p><strong>回避方法</strong></p>
<ul class="simple">
<li><p>なし</p></li>
</ul>
