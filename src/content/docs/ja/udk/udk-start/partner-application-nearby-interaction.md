<span id="anchor-ni-app"></span><h1>Qorvo の近くのインタラクション</h1>
<p>本节提供了 <a class="reference external" href="https://apps.apple.com/tr/app/qorvo-nearby-interaction/id1615369084">Qorvo Nearby Interaction</a> 的所有必要安装和设置步骤 应用程序。配备U1芯片的iPhone（型号：iPhone 11、iPhone 12和iPhone 13）需要iOS 16。</p>
<div class="section" id="installation-instructions">
<h2>インストール手順</h2>
<p>iOS 16 から、サードパーティ製アクセサリを使用したバックグラウンド モードを備えた Nearby Interaction が導入されました。iPhone を iOS 16 バージョン以降にアップデートする必要があります。 <a class="reference external" href="https://apps.apple.com/tr/app/qorvo-nearby-interaction/id1615369084">Qorvo Nearby Interaction</a> iOS アプリは、Apple App Store からダウンロードできます:</p>
<blockquote>
<div><div class="figure align-center" id="id6">
<a class="reference internal image-reference" href="../../../_images/qr-code-qorvo-nearby-interaction.png"><img alt="../../../_images/qr-code-qorvo-nearby-interaction.png" src="/docs-assets/ja/_images/qr-code-qorvo-nearby-interaction.png" style="width: 229.60000000000002px; height: 229.60000000000002px;"></a>
<p class="caption"><span class="caption-text"><a class="reference external" href="https://apps.apple.com/tr/app/qorvo-nearby-interaction/id1615369084">Apple App Store</a> にアクセスします</span></p>
</div>
</div></blockquote>
<p>インストール後、iOS デバイスのアイコンをタップして、<a class="reference external" href="https://apps.apple.com/tr/app/qorvo-nearby-interaction/id1615369084">Qorvo Nearby Interaction</a> アプリを開きます。</p>
