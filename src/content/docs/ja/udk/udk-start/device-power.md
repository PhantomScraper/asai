<span id="id1"></span><h1>電源オプション</h1>
<p>UDK デバイスは、USB-C ケーブルまたは 3.7V バッテリー電源という 2 つの実行可能な電源オプションを提供します。</p>
<a class="reference internal image-reference" href="../../../_images/powering_by_usb_battery.png"><img alt="../../../_images/powering_by_usb_battery.png" class="align-right" src="/docs-assets/ja/_images/powering_by_usb_battery.png" style="width: 171.20000000000002px; height: 282.40000000000003px;"></a>
<p><strong>USB タイプ C</strong></p>
<p>各デバイスには 2 つの <a class="reference internal" href="/docs/ja/udk/udk-start/device-hw-interfaces#hardware-interface"><span class="std std-ref">USB-C ポート</span></a> があります。どちらもボードの電源入力ポートとして機能するように設計されています。電源はPCまたは5Vアダプターから供給できます。必要に応じて、両方の <a class="reference internal" href="/docs/ja/udk/udk-start/device-hw-interfaces#hardware-interface"><span class="std std-ref">USB-C ポート</span></a> を同時に使用できます。</p>
<p><strong>バッテリー</strong></p>
<p>さらに、デバイスはバッテリーを使用して電力を供給できるため、ポータブル電源ソリューションが提供されます。必要に応じて、デバイスにはバッテリーと <a class="reference internal" href="/docs/ja/udk/udk-start/device-hw-interfaces#hardware-interface"><span class="std std-ref">USB-C ポート</span></a> から同時に電力を供給できます。</p>
