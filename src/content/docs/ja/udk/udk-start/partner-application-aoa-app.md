<span id="anchor-ranging-aoa"></span><h1>UWB レンジングおよび AoA デモ アプリ</h1>
<p>このセクションでは、Windows、macOS、および Linux プラットフォームへの UWB Ranging &amp; AoA Demo App GUI アプリケーションのインストールについて説明します。</p>
<div class="section" id="minimum-system-requirements">
<h2>最小システム要件</h2>
<div class="sd-tab-set docutils">
<input checked="checked" id="sd-tab-item-0" name="sd-tab-set-0" type="radio">
<label class="sd-tab-label" data-sync-id="1" for="sd-tab-item-0">
Windows の場合</label><div class="sd-tab-content docutils">
<ul class="simple">
<li><p>オペレーティング システム: Windows 7 以降</p></li>
<li><p>RAM: 2 GB RAM 以上</p></li>
</ul>
</div>
<input id="sd-tab-item-1" name="sd-tab-set-0" type="radio">
<label class="sd-tab-label" data-sync-id="2" for="sd-tab-item-1">
macOS の場合</label><div class="sd-tab-content docutils">
<ul class="simple">
<li><p>オペレーティング システム: macOS 10.13 以降</p></li>
<li><p>RAM: 2 GB 以上の RAM</p></li>
</ul>
</div>
<input id="sd-tab-item-2" name="sd-tab-set-0" type="radio">
<label class="sd-tab-label" data-sync-id="3" for="sd-tab-item-2">
Linux の場合</label><div class="sd-tab-content docutils">
<ul class="simple">
<li><p>オペレーティング システム: 最新のディストリビューションのほとんどが動作するはずです</p></li>
<li><p>RAM: 2 GB 以上の RAM</p></li>
</ul>
</div>
</div>
</div>
<div class="section" id="installation-instructions">
<h2>インストール手順</h2>
<div class="sd-tab-set docutils">
<input checked="checked" id="sd-tab-item-3" name="sd-tab-set-1" type="radio">
<label class="sd-tab-label" data-sync-id="1" for="sd-tab-item-3">
Windows の場合</label><div class="sd-tab-content docutils">
<ol class="arabic">
<li><p>インストール パッケージを準備します: UWB をダウンロードします <a class="reference external" href="https://drive.google.com/drive/u/1/folders/1cvBl6kUKOg8EnY-YCcn9iEIBrIlGIgzU">LEAPS Web サイト</a> からの測距および AoA デモ アプリのインストール パッケージ (.exe ファイル)。</p></li>
<li><p>パッケージ <code class="docutils literal notranslate"><span class="pre">UWB-Ranging-and-AoA-Demo-App-[x.y.z]-setup.exe</span></code> をダウンロードしたディレクトリに移動します。</p></li>
<li><p>インストーラーの画面上の指示に従って、インストールを完了します。</p></li>
<li><p>UWB レンジングと AoA デモ アプリをセットアップします。続行するには、<span class="red-text">次へ</span> ボタンを押してください。</p>
<div class="figure align-center" id="id3">
<a class="reference internal image-reference" href="../../../_images/image25.png"><img alt="../../../_images/image25.png" src="/docs-assets/ja/_images/image25.png" style="width: 396.5px; height: 177.0px;"></a>
<p class="caption"><span class="caption-text">UWB レンジングと AoA デモ アプリのセットアップ</span></p>
</div>
</li>
<li><p>次のステップでは、インストール ディレクトリを選択します。続行するには、<span class="red-text">次へ</span> ボタンを押してください。</p>
<div class="figure align-center" id="id4">
<a class="reference internal image-reference" href="../../../_images/image26.png"><img alt="../../../_images/image26.png" src="/docs-assets/ja/_images/image26.png" style="width: 398.0px; height: 179.0px;"></a>
<p class="caption"><span class="caption-text">インストールディレクトリの選択</span></p>
</div>
</li>
<li><p>次のステップが表示されます: "<span class="red-text">このアプリがデバイスに変更を加えることを許可しますか?</span>" この通知は、アプリが適切に機能するために必要な権限に関連しています。したがって、<code class="docutils literal notranslate"><span class="pre">はい</span></code> を選択してインストールを続行します。</p>
<div class="figure align-center" id="id5">
<a class="reference internal image-reference" href="../../../_images/image27.png"><img alt="../../../_images/image27.png" src="/docs-assets/ja/_images/image27.png" style="width: 241.0px; height: 191.0px;"></a>
<p class="caption"><span class="caption-text">このアプリがデバイスに変更を加えるのを許可しますか?</span></p>
</div>
</li>
<li><p>次のステップでは、コンポーネントを選択します。 <span class="red-text">UWB レンジング &amp; AoA デモ アプリ</span> を選択します。続行するには、<span class="red-text">次へ</span> ボタンを押してください。</p>
<div class="figure align-center" id="id6">
<a class="reference internal image-reference" href="../../../_images/image28.png"><img alt="../../../_images/image28.png" src="/docs-assets/ja/_images/image28.png" style="width: 397.0px; height: 177.0px;"></a>
<p class="caption"><span class="caption-text">コンポーネントの選択</span></p>
</div>
</li>
<li><p>次のステップでは、使用許諾契約書全体を読みます。同意する場合は、<span class="red-text">ライセンスに同意します</span> を選択します。続行するには、<span class="red-text">次へ</span> ボタンを押してください。</p>
<div class="figure align-center" id="id7">
<a class="reference internal image-reference" href="../../../_images/image29.png"><img alt="../../../_images/image29.png" src="/docs-assets/ja/_images/image29.png" style="width: 396.5px; height: 178.0px;"></a>
<p class="caption"><span class="caption-text">使用許諾契約</span></p>
</div>
</li>
<li><p>次のステップは、スタート メニューのショートカットです。名前を変更して、<span class="red-text">次へ</span> ボタンを押して続行します。</p>
<div class="figure align-center" id="id8">
<a class="reference internal image-reference" href="../../../_images/image30.png"><img alt="../../../_images/image30.png" src="/docs-assets/ja/_images/image30.png" style="width: 397.0px; height: 177.0px;"></a>
<p class="caption"><span class="caption-text">スタート メニューのショートカット</span></p>
</div>
</li>
<li><p>次のステップ、インストールの準備が整いました。続行するには、<span class="red-text">次へ</span> ボタンを押してください。</p>
<div class="figure align-center" id="id9">
<a class="reference internal image-reference" href="../../../_images/image311.png"><img alt="../../../_images/image311.png" src="/docs-assets/ja/_images/image311.png" style="width: 396.5px; height: 178.0px;"></a>
<p class="caption"><span class="caption-text">インストールの準備ができました。</span></p>
</div>
</li>
<li><p>次のステップでは、UWB レンジングと AoA デモ アプリをインストールします。続行するには、<span class="red-text">次へ</span> ボタンを押してください。</p>
<div class="figure align-center" id="id10">
<a class="reference internal image-reference" href="../../../_images/image32.png"><img alt="../../../_images/image32.png" src="/docs-assets/ja/_images/image32.png" style="width: 398.0px; height: 178.5px;"></a>
<p class="caption"><span class="caption-text">UWB レンジングと AoA デモ アプリのインストール</span></p>
</div>
</li>
<li><p>インストールプロセスが完了したら、アプリケーションを開始できます。</p>
<div class="figure align-center" id="id11">
<a class="reference internal image-reference" href="../../../_images/image33.png"><img alt="../../../_images/image33.png" src="/docs-assets/ja/_images/image33.png" style="width: 398.0px; height: 177.5px;"></a>
<p class="caption"><span class="caption-text">UWB レンジングと AoA デモ アプリのセットアップと開始プログラムの完了</span></p>
</div>
</li>
<li><p>インストールが完了すると、インストール構成に応じて、[スタート] メニューまたはコンピューター上に UWB Ranging &amp; AoA デモ アプリを配置して実行できます。</p>
<div class="figure align-center" id="id12">
<a class="reference internal image-reference" href="../../../_images/image34.png"><img alt="../../../_images/image34.png" src="/docs-assets/ja/_images/image34.png" style="width: 713.0px; height: 464.5px;"></a>
<p class="caption"><span class="caption-text">UWB レンジングおよび AoA デモ アプリのメイン画面</span></p>
</div>
</li>
</ol>
<p>これで、UWB Ranging および AoA デモ アプリが正常にインストールされ、起動されました。</p>
</div>
<input id="sd-tab-item-4" name="sd-tab-set-1" type="radio">
<label class="sd-tab-label" data-sync-id="2" for="sd-tab-item-4">
macOS の場合</label><div class="sd-tab-content docutils">
<ol class="arabic">
<li><p>インストール パッケージの準備: UWB Ranging および AoA デモ アプリのインストール パッケージ (DMG ファイル) を <a class="reference external" href="https://drive.google.com/drive/u/1/folders/1cvBl6kUKOg8EnY-YCcn9iEIBrIlGIgzU">LEAPS Web サイト</a>.</p></li>
<li><p>Mac でセキュリティ設定を構成します。 <span class="red-text">システム環境設定</span> で、<span class="red-text">セキュリティとプライバシー</span> をクリックし、次に <span class="red-text">一般</span> をクリックします。ロックをクリックしてパスワードを入力し、変更を加えます。ヘッダーの下にある <span class="red-text">App Store と識別された開発者</span> を選択します。 <span class="red-text">次からダウンロードされたアプリを許可します</span></p>
<div class="figure align-center" id="id13">
<a class="reference internal image-reference" href="../../../_images/image22.png"><img alt="../../../_images/image22.png" src="/docs-assets/ja/_images/image22.png" style="width: 664.0px; height: 582.0px;"></a>
<p class="caption"><span class="caption-text">macOS のセキュリティとプライバシー</span></p>
</div>
</li>
<li><p>パッケージ <code class="docutils literal notranslate"><span class="pre">UWB-Ranging-and-AoA-Demo-App-[x.y.z].dmg</span></code> をダウンロードしたディレクトリに移動します。</p></li>
<li><p>UWB レンジングおよび AoA デモ アプリは、DMG で直接利用できます。これを <span class="red-text">Applications</span> フォルダーにドラッグすると、インストールが完了します。</p>
<div class="figure align-center" id="id14">
<a class="reference internal image-reference" href="../../../_images/image23.png"><img alt="../../../_images/image23.png" src="/docs-assets/ja/_images/image23.png" style="width: 791.0px; height: 369.0px;"></a>
<p class="caption"><span class="caption-text">macOS に UWB レンジングと AoA デモ アプリをインストールする</span></p>
</div>
</li>
<li><p>インストールが完了すると、UWB Ranging および AoA デモ アプリが <span class="red-text">Applications</span> フォルダーで利用できるようになります。</p>
<div class="figure align-center" id="id15">
<a class="reference internal image-reference" href="../../../_images/image24.png"><img alt="../../../_images/image24.png" src="/docs-assets/ja/_images/image24.png" style="width: 203.0px; height: 167.0px;"></a>
<p class="caption"><span class="caption-text">インストールが完了すると、UWB Ranging および AoA デモ アプリが Applicationsフォルダーで利用できるようになります。</span></p>
</div>
</li>
<li><p>“UWB レンジング &amp; AoA デモ アプリ” を開きます。</p>
<div class="figure align-center" id="id16">
<a class="reference internal image-reference" href="../../../_images/image34.png"><img alt="../../../_images/image34.png" src="/docs-assets/ja/_images/image34.png" style="width: 713.0px; height: 464.5px;"></a>
<p class="caption"><span class="caption-text">UWB レンジングおよび AoA デモ アプリのメイン画面</span></p>
</div>
</li>
</ol>
<p>これで、UWB Ranging および AoA デモ アプリが正常にインストールされ、起動されました。</p>
</div>

</pre></div>
</div>
