<span id="id1"></span><h1>PANS PRO UWBS</h1>
<p><a class="reference internal" href="#pans-pro-uwbs"><span class="std std-ref">PANS PRO UWBS</span></a> は、完全なリアルタイム ロケーション システムおよびネットワーク スタックであり、アンカー、タグ、またはゲートウェイ ノードに構成可能です。</p>
<hr class="docutils">
<div class="section" id="key-features">
<h2>主な機能</h2>
<ul class="simple">
<li><p>TWR (双方向レンジング) 技術を利用: 統合された位置エンジンにより、デバイスは TWR を使用したナビゲーション モードで独立して動作できます。</p></li>
<li><p>ネットワーク プロファイル 0 をサポート: 高容量と低消費電力を備えた完全にスケーラブルな <a class="footnote-reference brackets" href="#note-1" id="id2">1</a>。</p></li>
<li><p>ネットワーク視覚化ツールのサポート: Android の PANS PRO Manager、および Windows、Linux、および macOS プラットフォームの LEAPS Center WebApp と互換性があります。</p></li>
<li><p>ファームウェアのアップデートとメンテナンス: 簡単なアップデートと継続的なメンテナンスが容易になります。</p></li>
<li><p>互換性: PANS RTLS (フリーミアム バージョン) と完全に互換性があり、長期的なバグ修正サポートが含まれています。</p></li>
<li><p>優れた電源管理: バッテリー寿命の延長を保証します。</p></li>
</ul>
</div>
