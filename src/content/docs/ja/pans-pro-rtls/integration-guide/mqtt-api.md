<span id="pans-mqtt-api"></span><h1>MQTT API</h1>
<div class="admonition caution">
<p class="admonition-title">注意</p>
<p>これらの MQTT API は <a class="reference internal" href="/docs/ja/pans-pro-rtls/integration-guide/pans-pro-uwbs#pans-pro-uwbs"><span class="std std-ref">PANS PRO UWBS</span></a> にのみ適用できることに注意してください。ユーザーが <a class="reference internal" href="/docs/ja/leaps-rtls/integration-guide/leaps-uwbs#leapsuwbs"><span class="std std-ref">LEAPS UWBS</span></a> を使用している場合、MQTT メッセージ プロトコルは異なります。</p>
</div>
