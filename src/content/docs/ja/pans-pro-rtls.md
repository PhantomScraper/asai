<span id="id1"></span><h1>PANS PRO RTLS</h1>
<div class="sd-card sd-sphinx-override sd-mb-3 sd-shadow-sm docutils">
<div class="sd-card-body docutils">
<p class="sd-card-text"><span class="red-text-bold">PANS PRO は製品版としてご利用いただけます。お問い合わせ：</span> <a class="reference external" href="mailto:sales%40leaps.global">sales<span>@</span>leaps<span>.</span>global</a></p>
</div>
</div>
<p><a class="reference internal" href="#pans-pro-rtls"><span class="std std-ref">PANS PRO RTLS</span></a> は、<a class="reference external" href="https://www.leapslabs.com/">LEAPS</a> が <a class="reference external" href="https://www.qorvo.com/">Qorvo</a> (旧 Decawave) 向けに開発した PANS RTLS の製品版です。資産、人、機器をリアルタイムで高精度に追跡できる、成熟した改良型テクノロジーです。<a class="reference internal" href="#pans-pro-rtls"><span class="std std-ref">PANS PRO RTLS</span></a> は、TWR (双方向測距) 方式を用いて、固定アンカーネットワーク周辺のタグの位置を特定します。タグの位置は、サーバーに接続されたゲートウェイ経由、または API を介して追跡対象の資産または車両に直接設定できます。</p>
