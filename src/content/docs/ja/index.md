<span id="welcome"></span><h1>LEAPSのドキュメントへようこそ！</h1>
<p><a class="reference external" href="https://www.leapslabs.com/">LEAPS</a> の公式ドキュメントへようこそ。ここでは、 <a class="reference internal" href="/docs/ja/udk#udk-demo-kit"><span class="std std-ref">UDK DEMO KIT</span></a>, <a class="reference internal" href="/docs/ja/leaps-rtls#leaps-rtls"><span class="std std-ref">LEAPS RTLS</span></a>, <a class="reference internal" href="/docs/ja/pans-pro-rtls#pans-pro-rtls"><span class="std std-ref">PANS PRO RTLS</span></a> (Real-Time Location System) の各コンポーネントに関する包括的で詳細な説明をご覧いただけます。私たちのドキュメントは、私たちのシステムを理解し、効果的に利用するために必要な情報を提供する貴重なリソースとして機能します。</p>
<p>ドキュメンテーションに加え、私たちはお客様やユーザーのグローバルなコミュニティのための専用プラットフォームである <a class="reference external" href="https://forum.leapslabs.com/">LEAPS フォーラム</a> を提供しています。このフォーラムでは、ディスカッションに参加したり、質問を投げかけたり、システムの複雑さを深く掘り下げることができます。このフォーラムでは、他のユーザーと交流し、知識や経験を共有することができます。</p>
<p>私たちのシステムの可能性を最大限に引き出すために、私たちのドキュメントを調べたり、 <a class="reference external" href="https://forum.leapslabs.com/">LEAPSフォーラム</a> に積極的に参加したりすることをお勧めします。</p>

<div class="section" id="organization-of-the-documentation">
<h2>ドキュメントの構成</h2>
<p>我々は文書を6つのセクションに整理している：</p>
<ul class="simple">
<li><p><a class="reference internal" href="/docs/ja/leaps-solutions#leaps-solution"><span class="std std-ref">LEAPSソリューション</span></a> セクションは、LEAPSが提供する製品や Ultra-Wideband リアルタイムロケーションシステムに関する簡潔な情報を提供します。</p></li>
<li><p><a class="reference internal" href="/docs/ja/udk#udk-demo-kit"><span class="std std-ref">UDK DEMO KIT</span></a> セクションは、包括的な詳細を提供し、Ultra-Wideband オールインワンのデモ用即使用可能なキットで簡単に始めることができます。</p></li>
<li><p><a class="reference internal" href="/docs/ja/leaps-rtls#leaps-rtls"><span class="std std-ref">LEAPS RTLS</span></a> セクションでは、先進的な Ultra-Wideband ナビゲーション・トラッキング・システムであるLEAPS RTLS（リアルタイム・ロケーション・システム）を深く理解する。</p></li>
<li><p><a class="reference internal" href="/docs/ja/pans-pro-rtls#pans-pro-rtls"><span class="std std-ref">PANS PRO RTLS</span></a> セクションでは、レガシー Ultra-Wideband ナビゲーション・トラッキング・システムであるPANS PRO RTLS（リアルタイム・ロケーション・システム）を深く理解する。</p></li>
<li><p><a class="reference internal" href="/docs/ja/faq#common-faq"><span class="std std-ref">よくある質問</span></a> セクションでは、よくある質問（FAQ）への回答を提供しています。</p></li>
<li><p><a class="reference internal" href="/docs/ja/support#technical-support"><span class="std std-ref">サポート</span></a> セクションは貴重なリソースとして機能し、支援やガイダンスがすぐに利用できるようにテクニカルサポート情報を提供する。</p></li>
</ul>

<div class="section" id="about-us">
<h2>会社概要</h2>
<ul class="simple">
<li><p><a class="reference external" href="https://www.leapslabs.com/">LEAPS</a> はプラハ（チェコ共和国、EU）を拠点とする会社で、ウルトラワイドバンドをベースとしたリアルタイム・ロケーションおよびリアルタイム・テレメトリー・システムの構築と販売に注力している。</p></li>
<li><p>ウルトラワイドバンドを使用したシステム開発で10年以上の経験を持つLEAPSは、ウルトラワイドバンドチップを有用なソリューションに変換する最も革新的なチームの一つです。</p></li>
<li><p>私たちのチームは、プラハ、ベトナムのホーチミン市、ブラジルのペロタスから情熱を持って働くシニア組込みエンジニアで構成されています。</p></li>
<li><p><a class="reference external" href="https://www.leapslabs.com/">LEAPS</a> は人気のDWM1001CとPANS RTLSの生みの親です。</p></li>
<li><p><a class="reference external" href="https://www.leapslabs.com/">LEAPS</a> はRTLSハードウェアとソフトウェア、設計サービス、技術ライセンスの提供に注力しています。</p></li>
<li><p><a class="reference external" href="https://www.leapslabs.com/">LEAPS</a> RTLS は、幅広いユースケースにロケーションサービスを提供する先進的なソフトウェアのコレクションです。近接、ナビゲーション、トラッキングからリアルタイム・データ・テレメトリ、ルーティング・バックホールまで、幅広い機能をカバーする市場で最も先進的な完全組み込み型UWBサブシステムを提供します。GPSモジュールを想像してみてください。</p></li>
<li><p><a class="reference external" href="https://www.leapslabs.com/">LEAPS</a> は、企業が新たな価値を見出し、プロセス効率を改善し、安全性を高め、信頼性を向上させ、屋内をナビゲートし、プロセスを自動化するのに役立つ新しい次元を提供します。</p></li>
</ul>
