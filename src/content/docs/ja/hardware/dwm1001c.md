<h1>DWM1001C</h1>
<p>DWM1001 モジュールは、Decawave の DW1000 超広帯域 (UWB) トランシーバ IC をベースにしており、IEEE 802.15.4-2011 UWB 実装です。UWBおよびBluetooth®アンテナ、すべてのRF回路、Nordic Semiconductor nRF52832、モーションセンサーを集積しています。</p>
<div class="admonition note">
<p class="admonition-title">注釈</p>
<p>詳細については、<a class="reference external" href="https://www.qorvo.com/products/p/DWM1001C">Qorvo's DWM1001C page</a> をご覧ください。</p>
</div>
<hr class="docutils">
<div class="section" id="key-features">
<h2>主な機能</h2>
<ul>
<li><p>10cm以内の測距精度。</p></li>
<li><p>UWBチャンネル5プリント基板アンテナ (6.5 GHz)。</p></li>
<li><p>6.8 Mbps データレート IEEE 802.15.4-2011 UWB 準拠</p></li>
<li><p>ノルディックセミコンダクター nRF52832</p></li>
<li><p>Bluetooth® 接続</p></li>
<li><p>Bluetooth® チップアンテナ</p></li>
<li><p>モーションセンサー： 3 軸加速度センサー</p></li>
<li><p>低消費電力スリープモードに最適化された消費電流： &lt;15μA</p></li>
<li><p>電源電圧: 2.8 V～3.6 V</p></li>
<li><p>サイズ： 19.1 mm x 26.2 mm x 2.6 mm</p></li>
<li><p>DWM1001C とマークされたモジュールは、ETSI、FCC、ISED 規制に適合しています。</p>
<blockquote>
<div><ul class="simple">
<li><p>FCC ID: 2AQ33-DWM1001、IC: 23794-DWM1001。</p></li>
<li><p>このデータシートのモジュールマーキングセクションを参照してください。</p></li>
</ul>
</div></blockquote>
</li>
</ul>
</div>
