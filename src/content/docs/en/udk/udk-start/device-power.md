<span id="id1"></span><h1>Power Options</h1>
<p>The UDK devices offer two viable power supply options: a USB-C cable or a 3.7V battery source.</p>
<a class="reference internal image-reference" href="../../../_images/powering_by_usb_battery.png"><img alt="../../../_images/powering_by_usb_battery.png" class="align-right" src="/docs-assets/_images/powering_by_usb_battery.png" style="width: 171.20000000000002px; height: 282.40000000000003px;"></a>
<p><strong>USB Type C</strong></p>
<p>Each device features two <a class="reference internal" href="/docs/en/udk/udk-start/device-hw-interfaces#hardware-interface"><span class="std std-ref">USB-C Ports</span></a>. Both are designed to serve as a power input port for the board.
The power can be sourced from a PC or a 5V adapter. Both <a class="reference internal" href="/docs/en/udk/udk-start/device-hw-interfaces#hardware-interface"><span class="std std-ref">USB-C Ports</span></a> can be used at the same time if needed.</p>
<p><strong>Battery</strong></p>
<p>In addition, the device can be powered using a battery, thereby offering a portable power solution.
The device can be powered from the battery and the <a class="reference internal" href="/docs/en/udk/udk-start/device-hw-interfaces#hardware-interface"><span class="std std-ref">USB-C Ports</span></a> at the same time if needed.</p>
