<span id="anchor-ranging-aoa"></span><h1>UWB Ranging &amp; AoA Demo App</h1>
<p>This section describes the installation of the UWB Ranging &amp; AoA Demo App GUI application on Windows, macOS and Linux platforms.</p>
<div class="section" id="minimum-system-requirements">
<h2>Minimum System Requirements</h2>
<div class="sd-tab-set docutils">
<input checked="checked" id="sd-tab-item-0" name="sd-tab-set-0" type="radio">
<label class="sd-tab-label" data-sync-id="1" for="sd-tab-item-0">
On Windows</label><div class="sd-tab-content docutils">
<ul class="simple">
<li><p>Operating System: Windows 7 or later</p></li>
<li><p>RAM: 2 GB RAM or more</p></li>
</ul>
</div>
<input id="sd-tab-item-1" name="sd-tab-set-0" type="radio">
<label class="sd-tab-label" data-sync-id="2" for="sd-tab-item-1">
On macOS</label><div class="sd-tab-content docutils">
<ul class="simple">
<li><p>Operating System: macOS 10.13 or later</p></li>
<li><p>RAM: 2 GB of RAM or more</p></li>
</ul>
</div>
<input id="sd-tab-item-2" name="sd-tab-set-0" type="radio">
<label class="sd-tab-label" data-sync-id="3" for="sd-tab-item-2">
On Linux</label><div class="sd-tab-content docutils">
<ul class="simple">
<li><p>Operating System: Most modern distributions should work</p></li>
<li><p>RAM: 2 GB of RAM or more</p></li>
</ul>
</div>
</div>
</div>
<div class="section" id="installation-instructions">
<h2>Installation instructions</h2>
<div class="sd-tab-set docutils">
<input checked="checked" id="sd-tab-item-3" name="sd-tab-set-1" type="radio">
<label class="sd-tab-label" data-sync-id="1" for="sd-tab-item-3">
On Windows</label><div class="sd-tab-content docutils">
<ol class="arabic">
<li><p>Prepare the installation package: Download the UWB Ranging &amp; AoA Demo App installation package (.exe file) from the <a class="reference external" href="https://drive.google.com/drive/u/1/folders/1cvBl6kUKOg8EnY-YCcn9iEIBrIlGIgzU">LEAPS website</a>.</p></li>
<li><p>Navigate to the directory where you downloaded the package <code class="docutils literal notranslate"><span class="pre">UWB-Ranging-and-AoA-Demo-App-[x.y.z]-setup.exe</span></code>.</p></li>
<li><p>Follow the on-screen instructions provided by the installer to complete the installation.</p></li>
<li><p>Set up UWB Ranging &amp; AoA Demo App. Press the <span class="red-text">Next</span> button to continue.</p>
<div class="figure align-center" id="id3">
<a class="reference internal image-reference" href="../../../_images/image25.png"><img alt="../../../_images/image25.png" src="/docs-assets/_images/image25.png" style="width: 396.5px; height: 177.0px;"></a>
<p class="caption"><span class="caption-text">Setup UWB Ranging &amp; AoA Demo App</span></p>
</div>
</li>
<li><p>Next step, choose the installation directory. Press the <span class="red-text">Next</span> button to continue.</p>
<div class="figure align-center" id="id4">
<a class="reference internal image-reference" href="../../../_images/image26.png"><img alt="../../../_images/image26.png" src="/docs-assets/_images/image26.png" style="width: 398.0px; height: 179.0px;"></a>
<p class="caption"><span class="caption-text">Choose Installation Directory</span></p>
</div>
</li>
<li><p>The next step will appear: “<span class="red-text">Do you want to allow this app to make changes to your device?</span>” This notification is related to permissions that the app requires to function properly. So, choose YES to continue the installation.</p>
<div class="figure align-center" id="id5">
<a class="reference internal image-reference" href="../../../_images/image27.png"><img alt="../../../_images/image27.png" src="/docs-assets/_images/image27.png" style="width: 241.0px; height: 191.0px;"></a>
<p class="caption"><span class="caption-text">Do you want to allow this app to make changes to your device?</span></p>
</div>
</li>
<li><p>Next step, select components. select the <span class="red-text">UWB Ranging &amp; AoA Demo App</span>. Press the <span class="red-text">Next</span> button to continue.</p>
<div class="figure align-center" id="id6">
<a class="reference internal image-reference" href="../../../_images/image28.png"><img alt="../../../_images/image28.png" src="/docs-assets/_images/image28.png" style="width: 397.0px; height: 177.0px;"></a>
<p class="caption"><span class="caption-text">Select Components</span></p>
</div>
</li>
<li><p>Next step, read the entire license agreement. If you agree, select the <span class="red-text">I accept the license</span>. Press the <span class="red-text">Next</span> button to continue.</p>
<div class="figure align-center" id="id7">
<a class="reference internal image-reference" href="../../../_images/image29.png"><img alt="../../../_images/image29.png" src="/docs-assets/_images/image29.png" style="width: 396.5px; height: 178.0px;"></a>
<p class="caption"><span class="caption-text">License Agreement</span></p>
</div>
</li>
<li><p>Next step, start menu shortcuts. You can change the name and Press the <span class="red-text">Next</span> button to continue.</p>
<div class="figure align-center" id="id8">
<a class="reference internal image-reference" href="../../../_images/image30.png"><img alt="../../../_images/image30.png" src="/docs-assets/_images/image30.png" style="width: 397.0px; height: 177.0px;"></a>
<p class="caption"><span class="caption-text">Start Menu Shortcuts</span></p>
</div>
</li>
<li><p>Next step, ready to installing. Press the <span class="red-text">Next</span> button to continue.</p>
<div class="figure align-center" id="id9">
<a class="reference internal image-reference" href="../../../_images/image311.png"><img alt="../../../_images/image311.png" src="/docs-assets/_images/image311.png" style="width: 396.5px; height: 178.0px;"></a>
<p class="caption"><span class="caption-text">Ready to Install.</span></p>
</div>
</li>
<li><p>Next step, installing UWB Ranging &amp; AoA Demo App. Press the <span class="red-text">Next</span> button to continue.</p>
<div class="figure align-center" id="id10">
<a class="reference internal image-reference" href="../../../_images/image32.png"><img alt="../../../_images/image32.png" src="/docs-assets/_images/image32.png" style="width: 398.0px; height: 178.5px;"></a>
<p class="caption"><span class="caption-text">Installing UWB Ranging &amp; AoA Demo App</span></p>
</div>
</li>
<li><p>You can start the application once the installation process is finished.</p>
<div class="figure align-center" id="id11">
<a class="reference internal image-reference" href="../../../_images/image33.png"><img alt="../../../_images/image33.png" src="/docs-assets/_images/image33.png" style="width: 398.0px; height: 177.5px;"></a>
<p class="caption"><span class="caption-text">Completing the UWB Ranging &amp; AoA Demo App Setup and Start Program</span></p>
</div>
</li>
<li><p>Once the installation is complete, you can run the UWB Ranging &amp; AoA Demo App by locating it in the Start menu or on your computer, depending on the installation configuration.</p>
<div class="figure align-center" id="id12">
<a class="reference internal image-reference" href="../../../_images/image34.png"><img alt="../../../_images/image34.png" src="/docs-assets/_images/image34.png" style="width: 713.0px; height: 464.5px;"></a>
<p class="caption"><span class="caption-text">UWB Ranging &amp; AoA Demo App main screen</span></p>
</div>
</li>
</ol>
<p>So, you have successfully installed and started the UWB Ranging &amp; AoA Demo App.</p>
</div>

On macOS</label><div class="sd-tab-content docutils">
<ol class="arabic">
