<span id="udkdemos"></span><h1>Demos</h1>
<p>Once the <span class="red-text">hardware</span> and <span class="red-text">required tools</span> are prepared, this section will provide a comprehensive guide on setting up each demo in greater detail.</p>
<p>It will cover the following aspects:</p>
<ul class="simple">
<li><p><strong>Required hardware and software</strong>: A brief list of requirements and information for each of the demo.</p></li>
<li><p><strong>Node or network configuration</strong>: Device setup and configuration to achieve the required demo.</p></li>
<li><p><strong>Running basic demo</strong>: Quick guideline to perform the main features of the demo.</p></li>
<li><p><strong>Technology evaluation</strong>: Guide and tips for the user to evaluate the technology using the demo setup.</p></li>
<li><p><strong>Advanced options</strong>: Guide and tips for the user to access more advanced features and suggestions for the next steps beyond the evaluation.</p></li>
<li><p><strong>Troubleshooting</strong>: Troubleshooting guide.</p></li>
</ul>
<p>Please select a demo and follow the instructions below to set it up quickly and ready for evaluation.</p>
