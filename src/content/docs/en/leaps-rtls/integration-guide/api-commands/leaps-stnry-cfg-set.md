<span id="id1"></span><h1>leaps_stnry_cfg_set</h1>
<p>Writes configuration of the stationary detection done by the tag node.
The configuration can be written even if stationary detection is
disabled (see <a class="reference internal" href="/docs/en/leaps-rtls/integration-guide/api-commands/leaps-cfg-tag-set#leaps-cfg-tag-set"><span class="std std-ref">leaps_cfg_tag_set</span></a>). Writes internal nonvolatile memory so
it should be used carefully. New sensitivity setting takes effect
immediately if stationary mode is enabled. Default sensitivity is “HIGH”.</p>
