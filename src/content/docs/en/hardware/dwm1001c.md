<h1>DWM1001C</h1>
<p>The DWM1001 module is based on Decawave’s DW1000 Ultra Wideband (UWB) transceiver IC, which is an IEEE 802.15.4-2011 UWB implementation. It integrates UWB and Bluetooth® antenna, all RF circuitry, Nordic Semiconductor nRF52832 and a motion sensor.</p>
<div class="admonition note">
<p class="admonition-title">Note</p>
<p>To get more information, please visit the <a class="reference external" href="https://www.qorvo.com/products/p/DWM1001C">Qorvo’s DWM1001C page</a>.</p>
</div>
<hr class="docutils">
<div class="section" id="key-features">
<h2>Key Features</h2>
<ul>
<li><p>Ranging accuracy to within 10cm.</p></li>
<li><p>The UWB Channel 5 printed PCB antenna (6.5 GHz)</p></li>
<li><p>6.8 Mbps data rate IEEE 802.15.4-2011 UWB compliant</p></li>
<li><p>Nordic Semiconductor nRF52832</p></li>
<li><p>Bluetooth® connectivity</p></li>
<li><p>Bluetooth® chip antenna</p></li>
<li><p>Motion sensor: 3-axis accelerometer</p></li>
<li><p>Current consumption optimized for low power sleep mode: &lt;15μA</p></li>
<li><p>Supply voltage: 2.8 V to 3.6 V</p></li>
<li><p>Size: 19.1 mm x 26.2 mm x 2.6 mm</p></li>
<li><p>Modules marked DWM1001C are certified to ETSI, FCC and ISED regulations</p>
<blockquote>
<div><ul class="simple">
<li><p>FCC ID: 2AQ33-DWM1001, IC: 23794-DWM1001.</p></li>
<li><p>See the module marking section of this datasheet.</p></li>
</ul>
</div></blockquote>
</li>
</ul>
</div>
<div class="section" id="software-compatibility">
<h2>Software Compatibility</h2>
<p>It is compatible with both <a class="reference internal" href="/docs/en/leaps-rtls#leaps-rtls"><span class="std std-ref">LEAPS RTLS</span></a> and <a class="reference internal" href="/docs/en/pans-pro-rtls#pans-pro-rtls"><span class="std std-ref">PANS PRO RTLS</span></a>.</p>
</div>
